# TXT2PDF
**带图形界面的 TXT 到 PDF 一键转换器**

平时用来将一些个人文档可读化的自用小工具。还有非常多的BUG。

### 使用

最多支持【大章节标题】（页头）->【分章节标题】（蓝色背景）->【段落标题】（黄色背景）->【正文】四个文本级别。

目录页点击链接跳转到每个标题所在处，每页右上角点击链接跳转回目录。

### 注意

> 源txt文档内容必须符合如图格式（类似yaml语法）

> 文档编码必须是 UTF-8 格式

> 缩进只能使用制表符

> 目录长度超出一页则无法正常显示（BUG）
